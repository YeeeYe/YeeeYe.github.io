interpolate方法可以用来在动画执行的过程中，根据给定的起始、最终值，计算动画的每一步需要的值。

interpolate还有一个作用可以让我们模拟css的关键帧。

注意点：interpolate只能被动画属性调用，普通字段是不可以调用interpolate方法
